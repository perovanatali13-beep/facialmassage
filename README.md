# Самомассаж лица — лендинг

Одностраничный сайт-лендинг экспресс-курса по самомассажу лица Алины Салаватовой.

- **Стек:** Next.js 15 (App Router), React 19, обычный CSS (без Tailwind).
- **Без базы данных.** Весь контент лежит в `lib/content.ts`.
- **Дизайн** в редакторском стиле: шрифты Cormorant + Jost, нумерованные секции 01–08, тёплая палитра.
- **Форма заявки** (имя, email, телефон) отправляется на почту через Resend.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Настройка отправки заявок на почту

Заявки приходят на **perova.natali13@gmail.com**. Для этого нужен бесплатный ключ Resend:

1. Зарегистрируйтесь на [resend.com](https://resend.com) под почтой `perova.natali13@gmail.com`.
2. В разделе **API Keys** создайте ключ.
3. Скопируйте `.env.local.example` в `.env.local` и вставьте ключ в `RESEND_API_KEY`.

Без своего домена письма уходят от `onboarding@resend.dev` — этого достаточно, чтобы
получать заявки на свою почту (адрес аккаунта Resend).

> Если ключ не задан, сайт продолжит работать: заявка не потеряется, а запишется в
> лог сервера (но письмо не отправится).

## Деплой на Vercel

1. Залейте проект в репозиторий и импортируйте в [Vercel](https://vercel.com).
2. В **Settings → Environment Variables** добавьте:
   - `RESEND_API_KEY` — ваш ключ Resend
   - `LEAD_TO` — `perova.natali13@gmail.com` (необязательно, это значение по умолчанию)
3. Нажмите **Deploy**.
